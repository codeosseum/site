Competitive testing and programming platform.

Codeosseum on GitHub

Features

Competitive Testing
The overwhelming majority of online competitive evaluator platforms focus solely on a single aspect of the software process: programming. However, that's not the case with Codeosseum! Codeosseum provides testing-first game modes helping players develop their testing skills.
Real-Time Game Modes
Competing against highscores is boring, the real excitement lies in player-versus-player battles! That's the thought behind Codeosseum's real-time game modes featuring PvP matches with instant feedback.
Dedicated Servers
Instead of using the well-known queuing model, Codeosseum runs each match on its own dedicated server.
GraalVM-based Evaluation
Codeosseum's submission processing engine is powered by GraalVM. This allows for efficient execution of user-provided codes as well as sandboxing and AST-instrumentation.
Open Source Licensing
Being licensed under the Apache 2.0 Software License, you can freely run Codeosseum on-prem or even modify the source code.
